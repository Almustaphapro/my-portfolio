Responsive Admin Dashboard

This project is a modern, responsive Admin Dashboard web application designed to help users track, manage, and visualize application data efficiently. It simulates a real-world management system where administrators can monitor budgets, projects, workload, and application statuses from a centralized interface. The dashboard focuses on clean UI, usability, and performance, following modern frontend best practices.

Features:

  • Responsive Sidebar Navigation (mobile & desktop friendly)
  • Dashboard Metrics Cards (Budget, Projects, Workload, Total Hours)
  • Modal Forms
  • Interactive Alerts

Technologies Used:

HTML5, CSS3, JavaScript(ES6), Bootstrap 5, SweetAlert2
