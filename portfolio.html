Portfolio Website

A responsive personal portfolio website built with HTML, CSS, and JavaScript. This project demonstrates modern design practices, responsive layouts, and interactive UI features.

Features:

  • Responsive design for desktop, tablet, and mobile
  • Interactive navigation and scroll animations
  • Project showcase with links to live demos
  • Contact form integrated with Formspree

Technologies Used:

HTML5, CSS3, JavaScript, Boxicons, ScrollReveal, Typed.js
