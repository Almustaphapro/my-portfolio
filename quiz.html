A modern, responsive, single-page Quiz App

A modern, responsive, single-page Quiz App built with plain HTML, CSS, and ES6 JavaScript. This project includes a welcome screen, 60 shuffled questions, a 60-minute countdown timer, a progress bar, question counter, smooth UI animations, and a final result screen. Ideal as a portfolio project.

Features:

  • 60 professional multiple-choice questions
  • Auto-shuffle questions each session
  • 60-minute countdown timer
  • Question counter (e.g. Question 2 / 60)
  • Animated progress bar
  • Option highlighting for correct/incorrect answers
  • Lightweight: no frameworks or external dependencies
  • Mobile friendly and accessible UI

Technologies Used:

HTML5, CSS3, JavaScript.
