Real-Time Cryptocurrency Dashboard

A responsive dashboard that displays real-time cryptocurrency prices, market trends, and portfolio tracking with interactive charts.

Features:

  • Real-time cryptocurrency prices using CoinGecko API
  • Interactive charts with Chart.js
  • Portfolio simulation with profit/loss calculations
  • Responsive design (mobile-first approach)
  • Dark/light theme toggle
  • Search and filtering functionality
  • Clean, modern UI with smooth animations

Technologies Used:

HTML5, CSS3, JavaScript, Boxicons, ScrollReveal
